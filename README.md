# AI 工单 Copilot

这是一个面向学习 AI 应用开发的渐进式项目。

当前目标不是一次性把所有功能做完，而是先完成前期调研、确定整体框架和阶段路线，再在这个框架下逐步实现一个可演示、可讲解的 AI 工单与知识库 Copilot。

## 当前已确定方向

- 前端使用 `React 19`
- 后端使用 `Node.js + Express 5`
- 数据库使用 `SQLite`
- 大模型使用 `GLM-4.7-Flash`
- 首版不做微服务
- 首版不依赖 MCP

## 当前开发方式

先把项目总图定清楚，再按小步推进：

1. 先做前期调研
2. 再确定需求边界和整体架构
3. 再制定阶段路线
4. 最后按功能逐步实现
